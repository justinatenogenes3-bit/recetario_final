# Recetario Kitchen v3.0

## Instalación
1. Descomprime el ZIP
2. Ejecuta: npm install
3. Inicia: node server.js
4. Abre: http://localhost:3000

## Clave de administrador
Por defecto: PALMA-ADMIN-2025
Configurable con variable de entorno: ADMIN_KEY=tu-clave node server.js

## Novedades v3.0
- Múltiples administradores
- Flujo de aprobación de recetas y categorías
- Favoritos y Ver más tarde
- Tiempo real (SSE) — sin recargar la página
- Responsive para móvil, tablet y escritorio
- Notificaciones para admins y contribuyentes
- Recetas destacadas y recién agregadas
- Expiración automática de recetas sin favoritos
- Anti-duplicados de títulos y categorías
- Estadísticas avanzadas por categoría, temporada, dificultad
- Rate limiting y mayor seguridad
